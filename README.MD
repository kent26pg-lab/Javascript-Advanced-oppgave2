# 📖 Boklogg

En liten og enkel nettside for å føre opp hvilken bøker man har lest

## Funskjoner

- Legg til bøker med tittel, forfatter, sjanger, hvor mange sider en boks har og hvor mange sider du har lest, Legg til rangering også.
- Bøker lagres i localStorage og er der når man refresher siden
- Duplikatsjekk. Man kan ikke legge til samme boken flere ganger
- Slett bøker
- Legg til en bok som favoritt

## Teknologi

- HTML
- CSS
- Javascript
- localStorage

## Hvordan bruke

- Fylle inn tittel, forfatter, sjanger, gi rangering, Hvor mange sider boken er på og hvor mange sider du har lest av den.
- Trykk legg til
- bøkene vises i tabellen under
- Slett bøker fra tabellen med å trykke på Slett
- Legge til bøker som favoritter med å trykke på stjerne knappen
