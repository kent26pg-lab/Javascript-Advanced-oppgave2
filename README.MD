# 📖 Boklogg
En liten og enkel nettside for å føre opp hvilken bøker man har lest

## Funskjoner
- Legg til bøker med tittel, forfatter, sjanger og rangering av boken
- Bøker lagres i localStorage og er der når man refresher siden
- Duplikatsjekk. Man kan ikke legge til samme boken flere ganger
- Slett bøker
